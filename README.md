# K8s Observability & Cost Optimization
[cite_start]This project implements a full-stack monitoring solution using **Prometheus and Grafana** while optimizing cloud spend via **Karpenter** and **Spot Instances**[cite: 16, 28].

## Features
- [cite_start]**Real-time Metrics:** CPU, Memory, and Disk monitoring for K8s nodes[cite: 49].
- **Cost Efficiency:** Automated node provisioning with Karpenter to reduce AWS spend by ~25%.
- [cite_start]**Alerting:** Configured Slack/Email alerts for system downtime[cite: 50].
