# K8s Observability & Cost Optimization
This project implements a full-stack monitoring solution using **Prometheus and Grafana** while optimizing cloud spend via **Karpenter** and **Spot Instances**

## Features
**Real-time Metrics:** CPU, Memory, and Disk monitoring for K8s nodes.
**Cost Efficiency:** Automated node provisioning with Karpenter to reduce AWS spend by ~25%.
**Alerting:** Configured Slack/Email alerts for system downtime.
